## üìù Quiz Question Guidelines

### Good Quiz Questions Should:
- ‚úÖ Test understanding, not just memorization
- ‚úÖ Have clear, unambiguous wording
- ‚úÖ Include 4 options (one correct, three plausible distractors)
- ‚úÖ Provide helpful explanations
- ‚úÖ Cover key concepts from the module
- ‚úÖ Range from easy to moderate difficulty

### Question Types to Include:
1. **Conceptual** - Test understanding of concepts
2. **Application** - Apply knowledge to scenarios
3. **Analysis** - Compare and contrast concepts
4. **Problem-solving** - Solve simple problems
5. **Recall** - Remember key facts and definitions

## üöÄ Quick Start Guide

### To Add a Quiz to a Module:

1. **Open the unit file** (e.g., `/app/pages/ds/units/Unit1.tsx`)
2. **Find the module section** (e.g., case 1 for module 1)
3. **Add quiz questions array** before the return statement
4. **Add Quiz component** at the end of module content
5. **Test in browser** to ensure it works

